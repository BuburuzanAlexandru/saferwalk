About




SaferWalk is a web application designed to ensure your walk home safer while considering various inputs like the crime rate in an area.







Development




Our app was designed using a variety of tools. The website was built using tools like CSS, HTML and Bootstrap. The API was designed using Mapbox. While the database was set up using Firebase, Node js etc. whereas, the algorithm was designed using Python and some libraries like Pandas, OSMnx and a few more.







Motivation
